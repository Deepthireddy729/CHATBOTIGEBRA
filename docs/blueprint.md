# **App Name**: Luminous Echo

## Core Features:

- Persistent Input Field: Maintain a readily accessible input field for composing text queries, allowing for continuous interaction without needing to re-establish focus.
- Streaming AI Responses: Display AI-generated responses in real-time, providing a dynamic and engaging conversational experience, where each portion of text appears as it's generated by the AI model.
- Scrollable Chat History: Implement a scrollable interface for viewing previous interactions, enabling users to easily revisit and reference earlier parts of the conversation.
- New Chat Initiation: Offer a clear and accessible 'New Chat' option to start a fresh conversation, effectively clearing the existing context and preparing the AI for a new topic.
- Contextual Response: Generate AI responses that consider the chat history within a single chat session, creating more relevant and coherent replies. The AI model uses a tool to evaluate how much of the previous chat history to use for each turn of the conversation.

## Style Guidelines:

- Background color: Deep charcoal gray (#121212) for a modern, sleek dark theme.
- Primary color: Vibrant electric blue (#7DF9FF) for chat bubbles and active elements.
- Accent color: Soft lavender (#E6E6FA) as a subtle gradient for interactive elements to create a delicate vibrancy.
- Body and headline font: 'Inter' sans-serif, known for its clean, modern aesthetic, ensuring excellent legibility on screen for both headlines and body text.
- Use minimalist line icons for actions such as sending messages, starting new chats, and any additional interactive controls.
- Implement a clean and intuitive layout, with clearly defined sections for the input field, conversation history, and action buttons, ensuring ease of navigation.
- Incorporate smooth, subtle transitions and animations to enhance the user experience, such as the gradual appearance of AI-generated text and fluid scrolling of the chat history.
